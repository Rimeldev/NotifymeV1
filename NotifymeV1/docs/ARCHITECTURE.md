# Architecture

Architecture du système NotifyMe

## Vue d'ensemble

NotifyMe suit une architecture microservices modulaire.

## Services

- **api-gateway**: Point d'entrée principal
- **message-service**: Gestion des messages
- **ai-service**: Intelligence artificielle
- **user-service**: Gestion des utilisateurs
- **sms-service**: Alertes SMS

Pour plus de détails, voir `notifyme_architecture_structure.md`
